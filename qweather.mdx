## 获取天气预报

| 项目 | 内容 |
|------|------|
| **API** | `POST /api/v2/3f/getWeather/:appname` |
| **说明** | 获取天气预报数据 |

**参数：**

| 参数 | 位置 | 必填 | 说明 |
|------|------|------|------|
| `appname` | path | 是 | 应用名 |
| `Key` | body | 是 | 设备 API Key |
| `longitude` | body | 是 | 经度 |
| `latitude` | body | 是 | 纬度 |
| `days` | body | 否 | 预报天数（默认7，最大30） |
| `lang` | body | 否 | 语言 |
| `UTCoffset` | body | 否 | UTC 偏移量（小时，如 +8） |

## 获取城市信息

| 项目 | 内容 |
|------|------|
| **API** | `POST /api/v2/3f/getCity/:appname` |
| **说明** | 获取城市信息列表 |

**参数：**

| 参数 | 位置 | 必填 | 说明 |
|------|------|------|------|
| `appname` | path | 是 | 应用名 |
| `Key` | body | 是 | 设备 API Key |
| `location` | body | 是 | 内容(文字/经纬度) |
| `range` | body | 否 | 搜索范围 |
| `number` | body | 否 | 返回结果数量 |

**返回：**

```json
{
  "status": 200,
  "result": [
    {
      "name": "北京",
      "lat": "39.90",
      "lon": "116.41",
      "adm2": "北京",
      "adm1": "北京市",
      "country": "中国"
    }
  ]
}
```

## 获取天气指数

| 项目 | 内容 |
|------|------|
| **API** | `POST /api/v2/3f/getIndices/:appname` |
| **说明** | 获取天气生活指数预报 |
<Callout type="info">
新版Eternal已内置天气指数计算，无需请求此API
</Callout>
**参数：**

| 参数 | 位置 | 必填 | 说明 |
|------|------|------|------|
| `appname` | path | 是 | 应用名 |
| `Key` | body | 是 | 设备 API Key |
| `longitude` | body | 是 | 经度 |
| `latitude` | body | 是 | 纬度 |
| `days` | body | 否 | 预报天数：`1d`（默认）或 `3d` |
| `type` | body | 否 | 指数类型ID（`0`=全部） |

**返回：**

```json
{
  "status": 200,
  "result": [
    {
      "Date": "2025-06-07",
      "data": [
        {
          "name": "运动指数",
          "level": "3",
          "category": "较不宜",
          "text": "天气较好，但考虑天气寒冷..."
        }
      ]
    }
  ]
}
```

## 获取分钟级降水

| 项目 | 内容 |
|------|------|
| **API** | `POST /api/v2/3f/getMinutely/:appname` |
| **说明** | 获取分钟级降水预报 |
<Callout type="warn">
分钟级降水仅支持中华人民共和国
</Callout>
**参数：**

| 参数 | 位置 | 必填 | 说明 |
|------|------|------|------|
| `appname` | path | 是 | 应用名 |
| `Key` | body | 是 | 设备 API Key |
| `longitude` | body | 是 | 经度 |
| `latitude` | body | 是 | 纬度 |

**返回：**

```json
{
  "status": 200,
  "result": {
    "summary": "95分钟后雨就停了",
    "start": "2025-06-07 10:00",
    "end": "2025-06-07 12:00",
    "type": "rain",
    "minutely": ["0.15", "0.23", "0.21", "0.17", ...]
  }
}
```
